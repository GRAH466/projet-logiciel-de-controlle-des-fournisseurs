# Application de gestion des fournisseurs et des articles

## Auteur

Grah Lowa Vincent Ezechiel

## Description du projet

Cette application est un **système complet de gestion des fournisseurs et des articles** développé en **C#** avec **Visual Studio Code**.
Elle permet de gérer les informations des fournisseurs ainsi que les articles qu’ils fournissent ou retirent du magasin. L’objectif est de centraliser et faciliter la gestion des stocks et des fournisseurs pour une entreprise.

## Objectifs

* Enregistrer et gérer les informations des fournisseurs
* Suivre les articles ajoutés ou retirés par chaque fournisseur
* Faciliter le suivi des stocks du magasin
* Offrir une interface simple pour la gestion quotidienne

## Fonctionnalités principales

### Gestion des fournisseurs

* Ajouter un fournisseur
* Modifier les informations d’un fournisseur
* Supprimer un fournisseur
* Rechercher un fournisseur par nom ou ID

### Gestion des articles

* Ajouter un nouvel article fourni par un fournisseur
* Retirer un article du stock
* Modifier les informations d’un article
* Suivre les mouvements d’articles par fournisseur
* Afficher la liste complète des articles

## Technologies utilisées

* C#
* Visual Studio Code
* .NET (selon la version utilisée)
* Base de données (SQLite, SQL Server ou autre)

## Contexte

Ce projet a été réalisé dans le cadre d’un **projet académique ou professionnel** pour appliquer les concepts de **programmation orientée objet**, **gestion des données** et **suivi des stocks** en C#.
